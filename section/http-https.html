HTTP

HTTP (HyperText Transfer Protocol) adalah protokol komunikasi yang digunakan oleh web browser dan server untuk saling bertukar data. HTTP menggunakan port 80 secara default, dan data yang dikirimkan melalui HTTP tidak dienkripsi.

HTTPS

HTTPS (HyperText Transfer Protocol Secure) adalah versi aman dari HTTP. Ia menggunakan enkripsi melalui protokol SSL/TLS untuk memastikan bahwa data yang dikirim antara browser dan server aman. HTTPS menggunakan port 443.
