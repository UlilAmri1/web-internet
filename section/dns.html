DNS

DNS (Domain Name System) adalah sistem yang menerjemahkan nama domain menjadi alamat IP yang dapat dibaca oleh komputer. Tanpa DNS, kita harus menghafalkan alamat IP untuk mengakses situs web. Proses ini mencakup query dari browser ke DNS server untuk mencari alamat IP yang sesuai dengan domain yang diminta.
